2. Read `AGENTS.md` at the folder root for hard rules.
3. Read `PKM/Documents/branding/jeffrey-positioning.md` — Jeff's core narrative. Every piece of content must serve this positioning.

## Cold-start briefing rule

Fresh context. Hawkeye must give you: the content type (resume / LinkedIn / blog / bio / YouTube), the target audience or opportunity, and any raw material Jeff dropped (notes, voice memo summary, rough draft). If the brief is thin, ask one clarifying question.
