# us-24

UniFi Switch 24 — the core 24-port switch for the Davis home network. All UniFi devices and wired connections run through this switch. Factory reset 2026-06-13 but adoption into [[uck-g2-plus]] is failing.

## Uplink topology

- Uplink to [[usg-pro-4]] (LAN port)
- Downlink to [[ap-firstfloor-livingroom]]
- Downlink to [[ap-secondfloor-masterbedroom]]
- Downlink to [[uck-g2-plus]]

## Adoption status (2026-06-13)

**ADOPTION FAILED.** Factory reset was performed (held reset button until LED changed) but SSH authentication with default credentials (ubnt/ubnt) fails — meaning the reset did not fully clear stored credentials.

**Resolution pending:** hard reset procedure required:
1. Power off the switch
2. Hold reset button
3. Power on while holding reset
4. Hold until LEDs cycle
5. Release — wait 2 minutes
6. SSH to 192.168.1.137 as ubnt/ubnt and run: `set-inform http://192.168.1.12:8080/inform`
7. Adopt from [[uck-g2-plus]]

Sparky to complete this via laptop + USB-to-Ethernet adapter (no RJ45 on [[jeff-laptop]] natively).

Switch is passing traffic normally on all connected ports despite not being adopted. VLAN configs from the controller have NOT been pushed to switch ports — IoT/Guest isolation is only enforced at the USG firewall level, not at the switch port level.
